# Product Overview

KodeOps is an open source AI-powered SDLC automation tool that transforms ideas into structured GitHub backlogs in minutes. The marketing site showcases the CLI tool's capabilities and drives adoption.

## Key Value Propositions

- **Speed**: Reduces planning overhead by 95%
- **Open Source**: Free CLI tool available on GitHub
- **AI-Powered**: Generates complete project documentation and backlogs
- **GitHub Integration**: Creates structured issues with proper labels

## Target Audience

- Software development teams
- Product managers
- Engineering leaders
- Open source contributors

## Core Features Highlighted

- PRFAQ generation
- Market Research Documents (MRD)
- Product Requirements Documents (PRD)
- Use case and persona creation
- Architecture documentation
- Automated GitHub issue creation

## Brand Positioning

Modern, developer-focused tool that bridges the gap between ideas and implementation through intelligent automation.
